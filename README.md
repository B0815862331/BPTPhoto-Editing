# BPTPhoto Editing

โปรแกรมตัดรูปภาพและเล่นเพลง พร้อมระบบฟังเสียงคำสั่ง

## ฟีเจอร์หลัก

### 1. ตัดรูปเป็นวงกลม
- เลือกรูปภาพและตัดเป็นวงกลม
- ปรับขนาดและตำแหน่งวงกลมได้
- บันทึกผลลัพธ์เป็น PNG
- ระบบฟังเสียงสั่งงาน

### 2. แก้ไขรูปภาพขั้นสูง
- ซูมเข้า/ออก รูปภาพ
- วาดวงกลม (กด Shift + ลาก)
- วาดลูกศร (กด Alt + ลาก)
- เพิ่มและแก้ไขข้อความ
- Undo/Redo การทำงาน
- บันทึกผลลัพธ์

### 3. AI ลบแสงจ้า Pro
- ลดแสงจ้า (Highlight)
- ลดแสงสะท้อน (Glare)
- เพิ่มความคมชัด (Sharpen)
- ลดเงาดำ (Shadow)
- ลบกระจกสะท้อน (Diffuse)
- คลิกบนภาพเพื่อลบแสงเฉพาะจุด
- AI ลบแสงอัตโนมัติ

### 4. เครื่องเล่นเพลง
- เล่นเพลงจากไฟล์ MP3
- เพลย์ลิสต์ระบบ
- ปรับระดับเสียง
- ลากย้ายตำแหน่งได้
- ระบบฟังเสียงควบคุม

### 5. ระบบฟังเสียง
- สั่งงานด้วยเสียงภาษาไทย
- รองรับคำสั่งต่างๆ:
  - "บันทึก", "ลบ", "รีเซ็ต"
  - "ซูมเข้า", "ซูมออก"
  - "ย้าย" (ย้ายเครื่องเล่นเพลง)
  - "เบาเสียง", "เพิ่มเสียง"
  - "เปลี่ยนเพลง", "เพลงถัดไป"
  - "ขอรูป" (เลือกรูปภาพใหม่)

## วิธีการใช้งาน

1. **เปิดไฟล์ `index.html` ในเบราว์เซอร์**
2. **เลือกรูปภาพ** สำหรับหน้าที่ต้องการ
3. **ใช้เมาส์หรือคำสั่งเสียง** เพื่อทำงานต่างๆ
4. **บันทึกรูปภาพ** เมื่อเสร็จสิ้น

## คำสั่งเสียงที่รองรับ

### คำสั่งทั่วไป
- "บันทึก" - บันทึกรูปภาพ
- "ลบ" - ล้างรูปภาพหรือการเลือก
- "รีเซ็ต" - รีเซ็ตการตั้งค่า
- "ขอรูป" - เลือกรูปภาพใหม่
